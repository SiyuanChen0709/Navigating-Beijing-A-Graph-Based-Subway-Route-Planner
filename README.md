# Navigating-Beijing-A-Graph-Based-Subway-Route-Planner
This project is the assignment requirement for University of Toronto's course, CSC111, and is entirely designed and written by students. 
It is based on Python and uses the least squares method to select starting and ending points. 
BFS and Dijkstra algorithms are used to search for Beijing subway stations, and interactive visualization is achieved through the Tkinter library. 
For more information, please check the Project Report.
